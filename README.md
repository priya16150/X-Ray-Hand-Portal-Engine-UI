# 🖐️ X-Ray Hand Portal Engine UI

A futuristic real-time **X-Ray Portal Visualization Engine** built using **HTML5 Canvas**, **JavaScript**, and **MediaPipe Holistic**. The application tracks both hands and full-body pose landmarks through a webcam and generates an interactive sci-fi portal interface with neural particles, body scanning effects, holographic HUDs, and dynamic visualizations.

## ✨ Features

* Real-time Hand Tracking
* Full Body Pose Detection
* AI-powered Segmentation Mask
* Interactive Hand Portal System
* Neural Particle Body Visualization (10,000+ particles)
* Futuristic X-Ray Scan Effect
* Dynamic Energy Streams & Sparks
* Holographic HUD Interface
* Glitch & Error Simulation Effects
* 3D Perspective Grid Rendering
* Live Body Skeleton Mapping
* Responsive Fullscreen UI

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Canvas API
* MediaPipe Holistic
* MediaPipe Camera Utils

## 🚀 How It Works

1. The webcam captures live video.
2. MediaPipe Holistic detects:

   * Hand Landmarks
   * Pose Landmarks
   * Segmentation Mask
3. When both hands are visible and fingers move close together:

   * A futuristic portal is activated.
   * The body is reconstructed using thousands of neural particles.
   * X-ray scanning and holographic effects are rendered in real-time.
4. Dynamic HUD panels display simulated biometric and system data.

```bash
https://github.com/priya16150/X-Ray-Hand-Portal-Engine-UI
```

Open the project:


Run with any local server:

```bash
npx serve
```

or simply open `index.html` in a modern browser.

| ------------------------- | ------------------------ |

| Move hands                | Resize portal            |
| Move body                 | Updates neural body scan |
